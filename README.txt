Per l'esecuzione del codice è necessaria la libreria NumPy di Python, il cui sito è raggiungibile qui: https://www.numpy.org/

Per l'installazione della libreria, è possibile seguire le poche istruzioni presenti qui: https://scipy.org/install.html

Consigliata l'installazione via pip, tramite il comando 'pip install numpy', dopo aver installato il pacchetto PiP
